# Gestión del sistema (demo)

Página web simple para:

- Ingresar un **documento**
- Ver la **población** asociada
- Si hay una lista de datos numéricos, calcular la **media**

## Cómo usar

1. Abrí `index.html` en el navegador (doble click).
2. Probá con documentos de ejemplo:
   - `40123456`
   - `39111222`
   - `44555666`
   - `30000000`
3. Editá los registros en `data.js`.
